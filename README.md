# 💰 加密货币大冒险

一个Python文字冒险游戏。你是一个大学生，从1000元本金开始，在加密货币世界里闯荡。

## 运行方法

```bash
python3 game.py
```

不需要安装任何依赖，纯Python标准库。

## 游戏简介

你是一个普通大学生，每个月生活费1500。有一天，室友靠比特币赚了大钱，你心动了。

你拿着存了好久的1000块钱，踏入了加密货币的世界。

接下来会发生什么？取决于你的每一个选择。

## 游戏特点

- **5个章节**，每章3-5个选择，剧情分支丰富
- **8种不同结局**：暴富、破产、被割韭菜、大佬、钻石手、监管、教授、归零
- **真实加密货币知识**：BTC、ETH、空投、合约、DeFi、NFT、链上分析等
- **背包系统**：钱包、硬件钱包、交易所账户、NFT、各种经验物品
- **随机事件**：市场闪崩、空投发币、手机推送等，增加不确定性
- **存档/读档**：随时保存，下次继续
- **ASCII艺术界面**：复古风格的游戏界面

## 操作指南

| 输入 | 功能 |
|------|------|
| 数字 | 做选择 |
| `s`  | 随时存档 |
| `i`  | 查看背包 |
| `h`  | 显示帮助 |
| `q`  | 保存并退出 |

## 核心属性

- 💰 **余额** - 你的现金
- 📦 **持仓** - 你持有的加密货币
- 📊 **知识** - 对加密货币的理解程度（影响判断）
- ⭐ **声望** - 在圈子里的名气
- ⚡ **压力** - 炒币带来的精神压力
- 🎒 **物品** - 你收集的各种东西

## 结局一览

| 结局 | 条件 |
|------|------|

| 💀 破产 | 亏得裤衩都不剩 |
| 🥬 被割韭菜 | 被骗子骗光 |

## 存档位置

存档文件保存在游戏同目录下的 `savegame.json`。
